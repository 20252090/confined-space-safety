# 밀폐공간 안전관리 앱

맨홀 등 밀폐공간에서 작업하는 고위험 작업자의 상태를 **감시인(관리자)** 이 현장에서 실시간으로 모니터링하고 입출입·안전점검을 기록하기 위한 웹 앱입니다.

- 순수 **HTML / CSS / JavaScript** (프레임워크·빌드 의존성 없음)
- 데이터는 브라우저 **localStorage** 에 저장
- 모바일(스마트폰/태블릿) 화면 기준, 장갑 착용·야외 조작을 고려한 큰 버튼·고대비 UI

## 주요 기능

| 탭 | 내용 |
|----|------|
| **대시보드(관제 뷰)** | 하수관로 **내부 종단면도**(지표면·지층·심도축·맨홀 수직구/사다리·관로/물·관경·경사·관저고) 위에 작업자를 **심도 위치**에 마커로 표시. 좌상단 현장정보(내부 인원·상태 집계·대표 가스 O₂·H₂S·CO·LEL), 우상단 경고 패널(위험/주의/정상 카운트·경보등급 필터·경보 리스트). **작업자 선택 시 AR 글라스 카메라 뷰**(맨홀 내부 1인칭 POV + HUD: O₂ 게이지·가스 바·심박/체온/경과/심도·상태/경고·응답/퇴장). 가스·바이탈은 시뮬레이션 값 |
| **작업자** | 등록/수정/삭제, 입장/퇴장 버튼으로 입출입 시각 자동 기록 |
| **체크리스트** | 가스측정·환기·송기마스크·안전대·무전기·작업허가서 6항목 모두 완료 시 '작업 시작' 활성화 |
| **기록** | 날짜별 작업 일지, 경보 발생 이력 |
| **설정** | 안전확인 주기/유예, 가스 임계치, 가스 시뮬레이션 모드, 데이터 내보내기/초기화 |

## 안전 로직

- **안전 확인**: 입장 후 설정 주기(기본 5분)가 지나면 '안전확인 요청'(주의) → 유예(기본 60초) 초과 시 '미응답 경보'(위험). 카드의 `안전확인 응답` 버튼으로 해제.
- **가스 경보**: 임계치 초과 시 해당 작업자 카드가 빨간색 경보 상태로 전환되고 경보 이력에 기록됩니다.
- **SOS / 전체 대피**: 상단 SOS 버튼 → 확인 후 전체 대피 오버레이 발령 및 이력 기록.
- 가스 값은 현재 **시뮬레이션** 이며, 설정의 '경보 테스트' 모드로 위험 상황을 재현할 수 있습니다.

## 실행

정적 파일이므로 `index.html` 을 브라우저로 열면 됩니다.

### GitHub Pages 배포
1. 이 폴더를 저장소에 push
2. Settings → Pages → Branch 를 `main` / root 로 지정
3. `.nojekyll` 이 포함되어 있어 별도 설정 없이 그대로 서비스됩니다.

## 파일 구조
```
index.html   화면 구조(5개 탭 + 모달 + 경보 오버레이)
style.css    산업 안전 다크 테마 / 상태색 / 큰 터치영역
app.js       상태관리(localStorage) · 시뮬레이션 · 경보 로직
```
